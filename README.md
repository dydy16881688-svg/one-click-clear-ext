# 一键打开 / 验证器 / 清理小工具

一个 Chrome 扩充功能（Manifest V3），一个图标搞定客服每天的三件事：

- 🚀 **一键启动**：打开所有常用后台网址（可分类、自动建立分页群组），并**自动填入帐号密码**
- 🔐 **内建验证器**：从手机 Google Authenticator **批量汇入**（扫汇出 QR），动态码**一键复制**
- 🧹 **一键清理**：登出所有网站（含 Google 服务器端登出）+ 彻底清除浏览痕迹

## ✨ 主要功能

| 分页 | 功能 |
|------|------|
| 🚀 启动 | 一键开所有网址、依分类建立分页群组、自动填帐密、强制统一字体（正黑体 / Comic Sans） |
| 🔐 验证器 | 手机批量汇入 TOTP、动态码即时显示（平时蓝、剩 5 秒变红）、点卡片即复制 |
| 🧹 清理 | 登出所有网站、彻底清空设定档资料、打开 Google 官方登出页 |

## 🔒 安全

- **主密码加密**：帐密与验证器用 PBKDF2 + AES-GCM 加密存储（零知识，忘记只能重设）
- 打开插件 / 设置页需**主密码解锁**
- 换电脑、不同设定档都需重新输入主密码

## ☁️ 跨机同步

- 设定存 Chrome Sync（绑各人 Google 帐号）+ 固定扩充 ID
- 登入自己的 Google 帐号并开同步 → 设定自动跨机、跨重开机还原

## 📦 安装

1. 下载本仓库 ZIP（绿色 **Code → Download ZIP**）并**解压**到固定资料夹
2. Chrome 开 `chrome://extensions` → 打开右上角**开发者模式**
3. 点**载入未封装项目** → 选解压出来的资料夹
4. 到该插件 **详细资料 → 扩充功能选项** 进行第一次设置

详细图文说明见资料夹内 `使用说明.html`。

## 🔄 更新

双击资料夹内的 **`更新.bat`** → 自动下载最新版覆盖 → 再到 `chrome://extensions` 点该插件的**刷新 ↻** 生效。

## 🛠️ 技术

原生 JavaScript / HTML / CSS · Manifest V3 · Web Crypto（加密）· jsQR（QR 解码）· OffscreenCanvas（动态图标）· chrome.storage.sync（同步）

> 注：本仓库为公开，但**不含任何帐号密码 / 验证器密钥**——那些只存在各使用者的浏览器本机与其个人 Google 云端，不在代码里。
